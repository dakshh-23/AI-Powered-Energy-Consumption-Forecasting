# AI-Powered Energy Consumption Forecasting System (Nexus Energy AI)

An enterprise-grade smart-grid predictive system utilizing Multi-Layer Perceptrons (MLP Neural Networks) to forecast building energy load profiles and automatically manage solar grid offsets.

## 🛠️ Tech Stack Used
* **Data Science Engine:** Python, Pandas, NumPy, Scikit-Learn (MLP Regressor)
* **Backend Framework:** Flask REST API Server
* **Frontend Visualization:** Cyberpunk Dark-Theme Dashboard (Tailwind CSS, Lucide Icons, Chart.js)

## 📦 How to Run Locally
1. Clone the repository and install requirements.
2. Train the Neural Network model:
   ```bash
   python3 train.py

```

3. Launch the unified interactive production server:
```bash
python3 app.py

```


4. Open `http://127.0.0.1:5000/` in your browser to experience the dashboard.
